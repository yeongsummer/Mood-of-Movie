## 프로젝트 소개
> 코로나때문에 놀러다니지도 못하고... 집에서 힐링할 수 있는 방법 없을까?



간단하게 힐링하고 싶을 때 영화 감상 어떠세요?

보고 싶은 영화를 찾는데 너무 오래걸린다거나 막상 영화를 본 후 원하던 영화가 아니라서 실망하지 않도록

**Mood of Movie**가 쉽고 빠르게 당신이 원하는 분위기의 영화를 찾아드립니다!
